Bidirectional Communication

PubCon allows any embedded device to talk to another. Whether its a PIC32 or an arduino board, all you need to do is to plug and start connecting. This is composed of two components - SEND and CONNECT

SEND: This invokes the publish() API to send any message over a desired 
channel.
CONNECT: This invokes the subscribe() API to listen for messages from other embedded devices connected to the same channel.


Device Discovery

Device Discovery helps you check which devices are online. It uses the presence() API part of the subscribe method to do this. 

PRESENCE: This invokes the publish() API to send any message over a desired 
channel.

pubnub.subscribe({
    channel: 'button-reply',
    message: function(m){console.log(m)},
    presence: function(m){console.log(m)}
})
